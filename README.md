Prince Cinema 
Welcome to the Prince Cinema project! This is a complete movie theater booking website. It allows users to browse movies, select seats, and book tickets easily. It also includes an admin panel for the cinema staff to manage the shows and see all bookings.

Features
For Users (Customers)
Browse Movies: See what movies are currently playing and what's coming soon.
Search: Easily search for a specific movie.
User Accounts: Sign up, log in, and manage your account.
Seat Selection: Pick your exact seats using a visual theater map. It prevents you from booking seats that are already taken.
My Bookings: View a history of all the tickets you have booked.
Favorites: Save your favorite movies to watch later.
Mobile Friendly: Works perfectly on phones, tablets, and computers.
For Admins (Staff)
Admin Login: Secure login area just for staff.
Add Shows: Schedule new movies, set the time, date, hall, and ticket price.
Manage Shows: See a list of all scheduled shows and delete them if needed.
Manage Bookings: See a list of every ticket booked by customers.

Built With
This project uses modern web technologies (The MERN Stack):

Frontend (Website Look & Feel): React.js, Tailwind CSS (for styling)
Backend (Server & Logic): Node.js, Express.js
Database: MongoDB (to save users, movies, shows, and bookings)

Project Structure
The project is split into two main parts:

client/ (Frontend): Contains all the code for what the user sees in the browser.
server/ (Backend): Contains the API, database connection, and secure logic.

How to Run the Project Locally
If you want to run this project on your own computer, follow these simple steps:

Open a terminal and navigate to the server folder.
Run npm install to download dependencies.
Run npm run dev to start the backend server.
Open a second terminal and navigate to the client folder.
Run npm install to download dependencies.
Run npm run dev to start the website.
Enjoy booking your movies!
